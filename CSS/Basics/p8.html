Defining color values

Colores in css often specify in the following formats;

  1. a color keyword like -red,green,trasparent,etc...
  2. a HEX and HEXA value -like #333,#ffff,#edfacc,#32345581 ,the a value can be added last as 01-99
  3. an RGB values (red,green,blue)-like rgb(255,0,0) and rgba(25,35,45,0.4)
